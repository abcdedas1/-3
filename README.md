项目文档：月历生成器

介绍
月历生成器是一个基于C语言的程序，用于生成指定年份和月份的月历。用户输入年份和月份后，程序会输出该月的日历，并以日历形式展示。

软件安装
a. 下载代码：从GitHub仓库下载月历生成器的源代码。
b. 编译代码：使用C编译器（如gcc）将源代码编译为可执行文件。

gcc calendar.c -o calendar
c. 安装完成：生成可执行文件后，即可使用月历生成器。

使用说明
a. 运行程序：在命令行中执行以下命令运行月历生成器：

./calendar
b. 输入年份和月份：按照程序提示，依次输入年份和月份，并按回车键确认。
c. 输出结果：程序将生成并输出指定年份和月份的月历。
d. 退出程序：月历生成器完成任务后，程序将自动退出。

示例
下面是一个使用月历生成器的示例：

输入年份和月，注意两者之间要加回车并且只能输入1900年及之后的年份

上述示例中，用户输入了年份2023和月份10，程序生成了2023年10月的月历，并以日历形式输出。

注意事项
a. 输入合法性：请确保输入的年份在1900年及之后，否则程序会提示年份错误并退出。
b. 闰年判断：程序根据输入的年份判断是否为闰年，从而确定每个月的天数。
c. 日历格式：程序以星期日为一周的第一天，以日历形式输出。
